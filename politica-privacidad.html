Política de privacidad

Esta Política de Privacidad describe cómo se trata la información en este sitio web.

1. Datos personales

Este sitio web no recopila datos personales mediante formularios, registros ni bases de datos.

2. Contacto externo

El contacto con el usuario se realiza a través de servicios externos como WhatsApp y Telegram.

Estos servicios pueden recopilar datos según sus propias políticas de privacidad:

  • WhatsApp (Meta Platforms)
  • Telegram

3. Cookies

Este sitio web no utiliza cookies propias de análisis, seguimiento o publicidad.

4. Uso de la web

La navegación en este sitio es libre y no requiere registro.

5. Cambios en esta política

Esta política puede actualizarse en cualquier momento sin previo aviso para adaptarse a cambios técnicos o legales.

6. Contacto

Para cualquier duda, puedes contactar mediante WhatsApp o Telegram desde la página principal.
